Konversi Aritmatika

Urutan pengerjaan ditentukan oleh posisi operator.

Infix (Manusia) A + B * C
Prefix (Polish) + A * B C
Postfix (Reverse Polish) A B C * +

Derajat Kekuatan Operator:

1. ^ (Pangkat) - Kanan ke Kiri

2. * / (Kali/Bagi) - Kiri ke Kanan

3. + - (Tambah/Kurang) - Kiri ke Kanan

Trik Ujian Manual: Beri kurung penuh sesuai derajat (A + (B * C)). Pindahkan operator ke depan kurung untuk Prefix atau ke belakang kurung untuk Postfix, lalu hapus semua kurung!

Tree & BST

Aturan Saklek BST: Kiri < Root (Lebih Kecil)
Kanan > Root (Lebih Besar)

Metode Traversal (Penelusuran):

Pre-Order: Root → Kiri → Kanan
In-Order: Kiri → Root → Kanan

*Tips UAS: Hasil In-Order pada BST pasti berurutan dari kecil ke besar!

Post-Order: Kiri → Kanan → Root

Terminologi Penting:

• Root: Node paling atas.

• Leaf (Daun): Node tanpa anak.

• Height: Jumlah level dari root ke daun terdalam.
