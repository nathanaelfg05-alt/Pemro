Modul 01

Konversi Infix, Prefix, & Postfix

Notasi aritmatika dibedakan berdasarkan penempatan posisi operator terhadap operand (variabel/angka). Pemahaman penempatan stack sangat krusial untuk bagian coding ujian tertulis.

Infix (Standard)
A + B

Operator berada di tengah kedua operand.

Prefix (Polish)
+ A B

Operator dipindah ke depan kedua operand.

Postfix (Reverse Polish)
A B +

Operator dipindah ke belakang kedua operand.

Aturan Precedence (Derajat Operator)

Operator Deskripsi Arah Asosiatif
^ Pangkat (Tertinggi) Kanan ke Kiri
* / Perkalian / Pembagian Kiri ke Kanan
+ - Penjumlahan / Pengurangan Kiri ke Kanan
Trik Cepat Manual (Metode Tanda Kurung)

Selesaikan ekspresi dengan memberi tanda kurung ketat berdasarkan urutan derajat matematika:
Kasus: A + B * C menjadi (A + (B * C)).

  • Mencari Prefix: Pindahkan operator ke sisi kiri setiap kurung pasangannya → + A * B C
  • Mencari Postfix: Pindahkan operator ke sisi kanan setiap kurung pasangannya → A B C * +
Modul 02

Tree & Binary Search Tree (BST)

Prinsip Binary Search Tree (BST):

Struktur data hirarki bercabang dua di mana setiap simpul anak di sebelah KIRI selalu bernilai lebih kecil dari Parent, dan simpul anak di sebelah KANAN selalu bernilai lebih besar.

Aturan Penyisipan:
- Node < Current: Masuk Kiri
- Node > Current: Masuk Kanan

Algoritma Penelusuran (Tree Traversal):

Pre-Order
Root → Kiri → Kanan

Cetak data sesaat setelah node dikunjungi.

In-Order
Kiri → Root → Kanan

*UAS Hack: Hasil traversal ini di BST pasti otomatis berurutan menaik!

Post-Order
Kiri → Kanan → Root

Root utama selalu dicetak pada urutan paling akhir.

Modul 03

Graph Traversal (BFS & DFS)

Penelusuran graf digunakan untuk mengunjungi simpul-simpul berjejaring. Karakteristik utama yang sering keluar di pilihan ganda UAS adalah perbedaan Struktur Data Bantu internalnya.

Breadth-First Search (BFS) Queue (FIFO)

Sistem penelusuran secara melebar horizontal. Mengunjungi seluruh tetangga terdekat yang terhubung langsung sebelum masuk ke tingkatan berikutnya.

Kegunaan: Mencari rute terpendek graf tanpa bobot.
Depth-First Search (DFS) Stack (LIFO)

Sistem penelusuran mendalam vertikal. Menelusuri satu cabang hingga menemukan jalan buntu (mentok), lalu mundur kembali (backtrack) ke cabang sebelumnya.

Kegunaan: Deteksi siklus graf, labirin topologi.
